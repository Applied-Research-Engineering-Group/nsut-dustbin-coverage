# NSUT Dustbin Coverage
This Flask-based dashboard works by dividing the NSUT campus into a grid of GPS-based zones and checking whether each zone lies within a specified walking distance of any dustbin. When dustbin coordinates are uploaded or automatically generated, the system converts all latitude and longitude points into a metric coordinate system and calculates accurate distances between each zone’s centroid and every dustbin. 
A zone is marked as “covered” if at least one bin falls within the chosen coverage radius. The application then compiles these results into a coverage percentage and presents them visually through an interactive Folium map, where covered zones appear in green, uncovered ones in red, and suggested new dustbin locations are highlighted for easy interpretation.

Using this method, we were able to perform a comprehensive accessibility audit of dustbins across the NSUT campus. By fine-tuning parameters such as zone size, coverage radius, and the number of dustbins, the model produced a realistic representation of how effectively the campus is serviced. The final results indicated that 90.2% of all campus zones fall within an appropriate walking distance of a dustbin, demonstrating that the majority of NSUT’s area is currently well-covered and meets accessibility standards for waste disposal.
